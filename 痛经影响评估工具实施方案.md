# 痛经影响评估工具实施方案

## 📋 项目概述

### 项目目标
实现一个完整的痛经影响评估工具，包含：
- **症状评估模块**：5个专业问题，评估疼痛程度、持续时间、位置、伴随症状、缓解偏好
- **职场影响评估模块**：4个专业问题，评估注意力、缺勤、沟通、支持需求
- **专业评分系统**：百分制评分，颜色编码（红色严重、橙色中等、绿色轻微）
- **个性化建议系统**：即时缓解、长期管理、职场支持建议
- **完整国际化支持**：中英文双语无缝切换

### 技术栈
- **前端框架**：Next.js 14 + React 18
- **开发语言**：TypeScript
- **样式框架**：Tailwind CSS
- **国际化**：next-intl
- **状态管理**：React Hooks
- **数据存储**：本地存储 + 状态管理

### 参考代码基础

### 1. 症状评估模块（基于参考代码 lIDgMx5）
```
问题1: 疼痛强度评估 (painLevel)
- 轻度 (1-3/10): 能感觉到，但不影响我的日常活动
- 中度 (4-6/10): 疼痛会干扰我，影响我的注意力和工作效率。可能需要非处方止痛药
- 重度 (7-8/10): 疼痛很强烈，我需要躺下或停止正在做的事情，难以正常活动
- 极重度 (9-10/10): 疼痛使人衰弱，难以忍受。我经常卧床不起，并可能伴有恶心或昏厥等其他症状

问题2: 疼痛持续时间 (painDuration)
- 在第一天持续几个小时
- 在经期的前1-2天内疼痛比较严重
- 持续3天或更长时间
- 疼痛不可预测，每个周期的差异很大

问题3: 疼痛位置 (painLocation) - 多选
- 下腹部（肚脐以下）
- 下背部（腰部）
- 放射到大腿上部
- 整个盆腔区域普遍的钝痛
- 身体一侧（左侧或右侧）

问题4: 伴随症状 (accompanyingSymptoms) - 多选
- 疲劳或筋疲力尽
- 头痛或偏头痛
- 恶心或呕吐
- 腹泻或便秘
- 易怒或情绪波动
- 腹部胀气
- 乳房胀痛
- 头晕或眩晕

问题5: 缓解偏好 (reliefPreference)
- 我需要针对当前疼痛的快速缓解方法
- 我更喜欢自然的或非药物的疗法
- 我想探索能够减少未来疼痛的长期管理策略
- 我愿意了解相关的医疗方案和专业建议
```

### 2. 职场影响评估模块（基于参考代码 lIDgMx5）
```
问题1: 注意力影响 (concentration)
- 对我的注意力没有影响
- 会稍微分散注意力，但我能应付
- 让我很难专注于任务
- 几乎无法集中注意力或保持工作效率

问题2: 缺勤情况 (absenteeism)
- 从未因经期症状而缺勤
- 很少（例如，每年一次或更少）
- 有时（每年几次）
- 经常（在大多数或所有周期中）

问题3: 沟通舒适度 (communication)
- 非常自在，我相信他们会支持
- 有些犹豫，不确定他们会作何反应
- 非常不自在，我会避免进行这种沟通
- 不适用 / 我没有上司

问题4: 支持需求 (support) - 多选
- 弹性的上下班时间
- 可选择在家工作
- 有安静的休息区可供使用
- 对月经健康有更开放和理解的文化氛围
- 有明确的经期休假政策
- 我不需要任何特定的支持
```

### 3. 专业评分系统（基于参考代码 lIDgMx5）
```
症状评估逻辑：
- 严重程度判断：painLevel === 'severe' || painLevel === 'verySevere'
- 基于疼痛程度生成建议：
  * 轻度：温和伸展、定期轻度运动
  * 中度：热敷、非处方止痛药、周期追踪、饮食调整
  * 重度/极重度：热敷+休息、深呼吸技巧、医疗咨询、长期管理策略
- 基于偏好调整建议：
  * 自然疗法：姜茶、抗炎食物
  * 医疗方案：妇科咨询、排除潜在疾病

职场评分算法（百分制）：
- 注意力影响：无影响(33) / 轻微(20) / 困难(10) / 无法(0)
- 缺勤情况：从未(33) / 很少(20) / 有时(10) / 经常(0)
- 沟通舒适度：自在(34) / 犹豫(15) / 不自在(5) / 不适用(15)
- 总分：0-100分

职场档案分类：
- 支持性环境 (>75分): 工作能力基本未受影响，沟通自在
- 适应性环境 (40-75分): 工作受到中等影响，需要支持措施
- 挑战性环境 (<40分): 症状严重影响工作，需要寻求支持

建议生成逻辑：
- 支持性：保持现状，继续沟通
- 适应性：非正式沟通，寻求支持
- 挑战性：记录影响，研究政策，练习沟通技巧
```

```
即时缓解建议：
- 轻度：温和的伸展运动或短途散步

长期管理建议：
- 轻度：定期进行瑜伽或普拉提等轻度运动
- 中度：追踪您的周期以预测疼痛、探索饮食调整（如减少咖啡因和盐分）
- 重度：咨询医疗保健专业人员进行诊断、与医生讨论长期疼痛管理策略
- 自然偏好：饮用姜茶或甘菊茶、将抗炎食物（如姜黄、绿叶蔬菜）纳入饮食
- 医疗偏好：预约妇科医生，以排除子宫内膜异位症或肌瘤等潜在疾病

职场支持建议：
- 支持性环境：您的工作能力似乎基本未受影响，并且您能自在地沟通需求。这是一个很好的基础
- 适应性环境：您的工作受到中等程度的影响。确定关键的支持措施可以显著改善您的体验
- 挑战性环境：您的症状严重影响了您的工作。寻求支持非常重要
- 沟通帮助：练习围绕工作影响来组织对话，而不是个人细节
- 支持选项：根据您的偏好，考虑正式或非正式地请求支持
```

```typescript
// 症状评估算法（来自参考代码）
export function calculateSymptomImpact(answers: Record<string, any>) {
  const { painLevel, painDuration, reliefPreference } = answers;
  const isSevere = painLevel === 'severe' || painLevel === 'verySevere';
  let summary = [];
  const recommendations = { immediate: [], longTerm: [] };

  // 基于疼痛程度生成建议
  if (painLevel === 'mild') {
    recommendations.immediate.push('温和的伸展运动或短途散步');
    recommendations.longTerm.push('定期进行瑜伽或普拉提等轻度运动');
  } else if (painLevel === 'moderate') {
    recommendations.immediate.push('在腹部或下背部加热敷垫');
    recommendations.immediate.push('考虑使用非处方止痛药（请先咨询医生）');
    recommendations.longTerm.push('追踪您的周期以预测疼痛');
    recommendations.longTerm.push('探索饮食调整（如减少咖啡因和盐分）');
  } else { // severe or verySevere
    recommendations.immediate.push('使用热敷垫并在舒适的位置休息');
    recommendations.immediate.push('使用深呼吸或冥想技巧应对急性疼痛');
    recommendations.longTerm.push('咨询医疗保健专业人员进行诊断');
    recommendations.longTerm.push('与医生讨论长期疼痛管理策略');
  }

  // 基于偏好调整建议
  if (reliefPreference === 'natural') {
    recommendations.immediate.unshift('饮用姜茶或甘菊茶');
    recommendations.longTerm.push('将抗炎食物（如姜黄、绿叶蔬菜）纳入饮食');
  }
  if (reliefPreference === 'medical') {
    recommendations.longTerm.unshift('预约妇科医生，以排除子宫内膜异位症或肌瘤等潜在疾病');
  }

  return { isSevere, summary, recommendations };
}

// 职场评分算法（来自参考代码）
export function calculateWorkplaceImpact(answers: Record<string, any>) {
  let score = 0;
  const suggestions = [];
  let profile = '';

  // 注意力影响评分
  if (answers.concentration) {
    switch (answers.concentration) {
      case 'none': score += 33; break;
      case 'slight': score += 20; break;
      case 'difficult': score += 10; break;
      case 'impossible': score += 0; break;
    }
  }

  // 缺勤情况评分
  if (answers.absenteeism) {
    switch (answers.absenteeism) {
      case 'never': score += 33; break;
      case 'rarely': score += 20; break;
      case 'sometimes': score += 10; break;
      case 'frequently': score += 0; break;
    }
  }

  // 沟通舒适度评分
  if (answers.communication) {
    switch (answers.communication) {
      case 'comfortable': score += 34; break;
      case 'hesitant': score += 15; break;
      case 'uncomfortable': score += 5; break;
      case 'na': score += 15; break;
    }
  }
  
  score = Math.round(score);

  // 职场档案分类
  if (score > 75) {
    profile = 'supportive';
    suggestions.push('您的工作能力似乎基本未受影响，并且您能自在地沟通需求。这是一个很好的基础。');
  } else if (score > 40) {
    profile = 'adaptive';
    suggestions.push('您的工作受到中等程度的影响。确定关键的支持措施可以显著改善您的体验。');
    suggestions.push('考虑与信任的经理或人力资源代表进行非正式的交谈，讨论所面临的挑战。');
  } else {
    profile = 'challenging';
    suggestions.push('您的症状严重影响了您的工作。寻求支持非常重要。');
    suggestions.push('首先记录症状对工作的影响。将我们的完整报告作为个人工具。');
    suggestions.push('研究您公司现有的病假和弹性工作政策。');
  }

  return { score, profile, suggestions };
}
```
