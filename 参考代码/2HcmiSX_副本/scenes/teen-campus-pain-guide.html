青少年校园痛经指南

专为学生设计的经期健康管理方案

🎒 校园专用

在学校也能轻松应对经期不适，不让疼痛影响学习生活

不同年龄段的特点与应对

初

初中阶段 (12-15岁)

月经初潮，身体适应期

生理特点

  • • 月经周期不规律
  • • 痛经可能较重
  • • 身体正在适应激素变化
  • • 情绪波动较大

重点关注

  • • 建立正确的经期认知
  • • 学习基本的自我护理
  • • 与家长开放沟通
  • • 关注营养和睡眠
高

高中阶段 (15-18岁)

周期趋于稳定，学习压力大

生理特点

  • • 月经周期逐渐规律
  • • 痛经程度相对稳定
  • • 身体发育基本成熟
  • • 对疼痛认识更清楚

重点关注

  • • 平衡学习与健康
  • • 学会压力管理
  • • 独立处理经期问题
  • • 建立健康的生活方式
大

大学阶段 (18-22岁)

独立生活，自主管理

生理特点

  • • 月经周期稳定
  • • 痛经模式固定
  • • 荷尔蒙水平相对平衡
  • • 身体完全发育

重点关注

  • • 完全自主管理
  • • 考虑专业治疗
  • • 维持健康生活习惯
  • • 为未来生活做规划

校园痛经管理策略

📚 课堂应对技巧

立即缓解方法

🍃
深呼吸练习

课堂上不被察觉的缓解方法，通过腹式呼吸放松肌肉

🪑
调整坐姿

使用软垫，尝试不同坐姿找到最舒适的位置

🤲
隐蔽按压

用手掌轻压下腹部，或按压手部穴位

🌡️
暖宝宝使用

贴在衣服内侧，提供持续的温热缓解

课堂注意事项

提前准备：

经期前在书包里准备暖宝宝、止痛药等应急用品

合理请假：

疼痛严重时不要硬撑，及时向老师说明情况

选择座位：

如可能，选择靠近门口或过道的座位，方便离开

课前服药：

在医生指导下，可在疼痛出现前预防性服药

🏫 善用校园设施

医务室/保健室

  • • 寻求专业建议
  • • 获得基础药物
  • • 短暂休息场所
  • • 了解健康知识

宿舍/休息室

  • • 充分休息调整
  • • 使用热敷设备
  • • 放松情绪
  • • 补充营养

食堂/小卖部

  • • 选择温热食物
  • • 购买生理用品
  • • 补充热水
  • • 避免生冷食物

💬 沟通指南

与老师沟通

沟通要点
  • • 选择合适的时机（私下交流）
  • • 简洁说明情况，无需过度详细
  • • 表达对学习的重视
  • • 询问补课或作业安排
话术建议

"老师，我身体不太舒服，可能需要休息一下。 今天的课程内容我会课后补上，请问有什么需要特别注意的吗？"

与家长沟通

沟通内容
  • • 描述疼痛具体症状
  • • 说明对学习生活的影响
  • • 请求就医或购买用品
  • • 寻求情感支持
克服羞涩
  • • 记住这是正常的生理现象
  • • 家长会理解和支持
  • • 及时沟通有助于获得帮助
  • • 可先与信任的人（如妈妈）谈话

同伴支持与互助

👭 朋友间的相互支持

情感支持

与信任的朋友分享感受，获得理解和鼓励

实际帮助

学习协助

校园应急包准备

🎒 书包常备物品

生理用品

  • • 卫生巾/棉条（2-3片）
  • • 小包装湿巾
  • • 密封袋（用于废物处理）
  • • 备用内裤

缓解用品

  • • 暖宝宝（2-3个）
  • • 非处方止痛药
  • • 小包装红糖
  • • 薄荷糖（缓解恶心）

🏠 宿舍储备

护理用品

  • • 热水袋/电热毯
  • • 充足的生理用品
  • • 换洗衣物
  • • 舒适的睡衣

营养补充

  • • 红糖、蜂蜜
  • • 红枣、桂圆干
  • • 即热营养粥
  • • 维生素补充剂
