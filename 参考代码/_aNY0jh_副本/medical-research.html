痛经类型的医学区分

医学定义概述

痛经（Dysmenorrhea）是指与月经相关的盆腔疼痛，根据 世界卫生组织（WHO） 和 美国妇产科医师学会（ACOG） 的标准分类， 痛经主要分为原发性痛经和继发性痛经两大类。这一分类对于制定正确的治疗策略具有重要临床意义。

🟢

原发性痛经

医学定义

原发性痛经是指无明显盆腔器质性病变的功能性痛经， 占所有痛经病例的90-95%。 根据Mayo Clinic的定义， 这是一种由正常生理过程引起的疼痛。

流行病学特征

  • 发病年龄：通常始于初潮后6-12个月，当排卵周期建立后
  • 患病率：在青少年女性中高达60-93%
  • 严重程度：约15-20%的患者症状严重到影响日常活动
  • 长期趋势：症状可能随年龄增长而减轻，特别是生育后

临床特征

  • • 疼痛性质：典型的痉挛性疼痛，位于下腹部正中
  • • 发作时间：月经来潮前数小时或月经第一天开始
  • • 持续时间：通常持续24-72小时
  • • 疼痛强度：在月经开始后24小时内达到峰值
  • • 伴随症状：恶心(89%)、呕吐(60%)、腹泻(60%)、头痛(45%)
🔴

继发性痛经

医学定义

继发性痛经是指由盆腔器质性疾病引起的痛经， 占所有痛经病例的5-10%。 WHO将其定义为"由可识别的盆腔病理状态引起的痛经"， 需要针对原发疾病进行治疗。

流行病学特征

  • 发病年龄：多见于25岁以后，特别是30-40岁年龄组
  • 病因分布：子宫内膜异位症占60-70%，子宫肌瘤占20-30%
  • 进展性：疼痛通常随时间加重
  • 生育影响：30-50%的继发性痛经患者存在生育问题

临床特征

  • • 疼痛性质：深部钝痛、胀痛，可能不规律
  • • 发作时间：月经前1-2周即可开始，持续时间更长
  • • 疼痛模式：呈进行性加重，对NSAIDs反应较差
  • • 非周期性疼痛：可在非经期出现盆腔疼痛
  • • 特殊症状：深部性交痛(77%)、不孕(30-50%)、异常出血(40%)

鉴别特征 原发性痛经 继发性痛经

病理生理机制深度解析

机制概述

根据 Mayo Clinic 和国际疼痛研究协会（IASP）的研究，原发性痛经的核心机制涉及前列腺素过度产生、 子宫肌肉过度收缩、血管收缩导致的局部缺血，以及疼痛感知系统的激活。

🧬 前列腺素的核心作用

前列腺素生成过程

  1. 磷脂酶A2激活：月经期子宫内膜细胞破裂，释放花生四烯酸
  2. 环氧合酶作用：COX-1和COX-2将花生四烯酸转化为前列腺素
  3. PGF2α和PGE2产生：这两种是引起痛经最重要的前列腺素
  4. 受体结合：前列腺素与子宫平滑肌受体结合

数据支持

  • 痛经患者月经期PGF2α浓度比无痛经者高3-7倍
  • PGE2浓度高2-4倍
  • 前列腺素浓度与疼痛强度呈正相关 (r=0.73)

前列腺素的作用效应

PGF2α的作用
  • • 引起强烈的子宫肌肉收缩
  • • 收缩子宫血管，导致局部缺血
  • • 直接刺激疼痛感受器
  • • 增加疼痛介质的敏感性
PGE2的作用
  • • 增强疼痛感受器敏感性
  • • 引起血管扩张和炎症
  • • 促进其他疼痛介质释放
  • • 影响中枢神经系统疼痛处理

💪 子宫收缩机制

正常收缩 vs 痛经收缩

正常收缩：
  • • 压力：50-80 mmHg
  • • 频率：3-4次/10分钟
  • • 持续时间：60-90秒
痛经收缩：
  • • 压力：120-200 mmHg
  • • 频率：6-8次/10分钟
  • • 持续时间：90-150秒

缺血机制

血流减少：强烈收缩时子宫血流可减少60-80%

缺血时间：持续缺血超过60秒开始产生疼痛

代谢产物：乳酸、腺苷等致痛物质累积

组织损伤：缺血再灌注损伤加重疼痛

神经机制

感受器激活：机械感受器和化学感受器同时激活

神经传导：通过腰骶神经丛传导至脊髓

脊髓处理：T10-L1节段疼痛信号整合

大脑处理：感觉皮层和情绪中枢共同参与

⚡ 其他疼痛介质

  • 白三烯：增强血管通透性，放大炎症反应
  • 血管加压素：促进子宫收缩，痛经患者水平升高50-70%
  • P物质：神经肽，传递疼痛信号并引起神经源性炎症
  • 组胺：增加血管通透性，引起组织水肿和疼痛加重

🧠 中枢敏化机制

  • 外周敏化：反复疼痛刺激使疼痛感受器敏感性增加
  • 脊髓风up：脊髓背角神经元兴奋性增强，疼痛放大
  • 下行抑制减弱：大脑下行疼痛抑制系统功能降低
  • 情绪影响：疼痛记忆增强焦虑和疼痛预期

临床诊断标准流程

诊断标准概述

根据 美国妇产科医师学会（ACOG） 和 世界卫生组织（WHO） 的临床指南， 痛经的诊断主要基于详细的病史采集、体格检查和必要的辅助检查。 准确的诊断是制定有效治疗方案的前提。

1

病史采集

详细疼痛史

  • • 疼痛发作时间和持续期
  • • 疼痛性质和强度（VAS评分）
  • • 疼痛位置和放射模式
  • • 诱发和缓解因素

月经史

  • • 初潮年龄和周期规律性
  • • 月经量和持续时间
  • • 痛经开始时间
  • • 症状变化趋势

其他症状

  • • 性交痛
  • • 排便或排尿困难
  • • 不孕史
  • • 伴随全身症状
2

体格检查

腹部检查

  • • 下腹部压痛和反跳痛
  • • 腹部包块触诊
  • • 肠鸣音听诊
  • • 腹部触诊技巧

盆腔检查 *

  • • 外阴和阴道视诊
  • • 宫颈检查和细胞学
  • • 双合诊检查子宫
  • • 附件区状况评估

* 青少年原发性痛经可能不需要盆腔检查

特殊体征

  • • 子宫活动度
  • • 触痛结节
  • • 子宫大小和形状
  • • 盆腔粘连征象
3

辅助检查

影像学检查

  • • 经腹/经阴超声：首选检查
  • • MRI：可疑内膜异位症
  • • CT：复杂盆腔病变
  • • 子宫输卵管造影：不孕评估

实验室检查

  • • CA-125：内膜异位症标志
  • • 炎症指标：CRP、ESR
  • • 感染筛查：阴道分泌物
  • • 激素水平：必要时检测

侵入性检查

  • • 宫腔镜：子宫内病变
  • • 腹腔镜：诊断金标准
  • • 组织活检：确诊需要
  • • MR增强：特殊情况下

现代医学治疗方案

治疗原则概述

根据 ACOG临床实践公告 和 Mayo Clinic治疗指南， 痛经治疗应采用阶梯式方法，从非药物治疗到药物治疗，从一线药物到二线药物， 个性化选择最适合患者的治疗方案。

药物治疗方案

1

NSAIDs（一线治疗）

推荐药物
  • • 布洛芬：400-800mg，每6-8h
  • • 萘普生：220-440mg，每8-12h
  • • 双氯芬酸：25-50mg，每8h
  • • 塞来昔布：200-400mg/日
有效性数据
  • • 总有效率：85-90%
  • • 疼痛缓解：VAS评分降低3-4分
  • • 起效时间：30-60分钟
  • • 预防性使用效果更佳
使用原则
  • • 疼痛开始前或刚出现时使用
  • • 规律给药比按需给药更有效
  • • 推荐饭后服用减少胃肠道刺激
  • • 一般连续使用3-5天
2

激素治疗

复合口服避孕药（COC）
  • • 机制：抑制排卵和内膜增生
  • • 有效率：60-90%
  • • 适应症：需要避孕的痛经患者
  • • 优势：同时解决多个问题
孕激素类
  • • LNG-IUS：宫内缓释系统，5年有效
  • • 地屈孕酮：10-20mg/日，周期后半期
  • • 甲羟孕酮：150mg肌注，每3个月
  • • 效果：减少月经量和痛经
选择考虑
  • • NSAIDs失败或不能耐受
  • • 需要避孕的年轻女性
  • • 月经量过多伴痛经
  • • 长期疼痛管理需求
3

其他药物

镇痛药物
  • • 对乙酰氨基酚：500-1000mg，每4-6h
  • • 曲马多：50-100mg，严重疼痛时
  • • 复合制剂：含咖啡因的复合镇痛剂
  • • 局部麻醉：利多卡因贴片
抗痉挛药物
  • • 间苯三酚：80-160mg，每8h
  • • 山莨菪碱：5-10mg，肌注或口服
  • • 氯化钾缓释片：缓解肌肉痉挛
  • • 镁剂：200-400mg元素镁/日
适用情况
  • • NSAIDs禁忌或过敏
  • • 激素治疗不适合
  • • 特殊情况下的症状性治疗
  • • 作为联合治疗的一部分

非药物治疗方法

🔥 物理治疗方法

热疗
  • • 有效性：与NSAIDs相当，联合使用效果更佳
  • • 机制：促进血流，放松肌肉，激活闸门控制理论
  • • 方法：热水袋、电热垫、温水浴（39-45°C）
  • • 累积证据：多项RCT证实有效
TENS（经皮神经电刺激）
  • • 有效性：中等效果，安全无副作用
  • • 参数：50-120Hz，20-30分钟/次
  • • 位置：下腹部或腰骶部
  • • 证据：Cochrane综述支持使用
按摩治疗
  • • 技术：腹部环形按摩、背部按摩
  • • 时间：每次15-20分钟
  • • 频率：疼痛期间每日2-3次
  • • 效果：减轻疼痛强度约30-40%

🧘 替代和补充疗法

针灸治疗
  • • 穴位：三阴交、关元、子宫、太冲
  • • 疗程：每周2-3次，连续3个月经周期
  • • 证据：多项研究显示中等效果
  • • 机制：调节神经内分泌系统
瑜伽和运动
  • • 推荐体式：猫牛式、婴儿式、蝴蝶式
  • • 有氧运动：中等强度，每周150分钟
  • • 效果：长期练习可减轻疼痛50-60%
  • • 机制：肌内啡肽释放，肌肉放松
心理治疗
  • • 认知行为疗法：改变疼痛认知模式
  • • 放松训练：深呼吸、渐进性肌肉放松
  • • 正念冥想：提高疼痛耐受性
  • • 压力管理：减少心理因素对疼痛的放大

常用药物作用机制与注意事项

💊 NSAIDs详细机制与临床应用

作用机制

主要机制：COX抑制
  • • COX-1抑制：减少基础前列腺素产生
  • • COX-2抑制：减少炎症性前列腺素
  • • PGF2α下降：减少子宫收缩强度
  • • PGE2下降：降低疼痛敏感性
次要机制
  • • 血管效应：改善子宫血流供应
  • • 神经调节：影响疼痛传导通路
  • • 炎症抑制：减少局部炎症反应
  • • 血小板功能：影响血液粘度

副作用与禁忌

常见副作用
  • • 胃肠道：恶心(8-15%)、胃痛(5-12%)、消化不良
  • • 中枢神经：头痛(2-8%)、头晕(1-5%)
  • • 皮肤：皮疹(<1%)、过敏反应
  • • 肾脏：水钠潴留(罕见)
禁忌症
  • • 绝对禁忌：活动性消化性溃疡、严重肝肾功能不全
  • • 相对禁忌：哮喘、心血管疾病、高血压
  • • 妊娠期：孕晚期禁用，可能导致胎儿动脉导管收缩
  • • 过敏史：对NSAIDs或阿司匹林过敏
临床使用要点
  • • 首选药物：布洛芬，安全性和有效性平衡最好
  • • 预防性使用：月经前24-48小时开始使用
  • • 联合治疗：可与对乙酰氨基酚联用增效
  • • 监测指标：胃肠道症状、肾功能、血压
  • • 停药指征：出现严重胃肠道反应或过敏
  • • 特殊人群：老年患者、肾功能不全者减量

🧪 激素类药物机制与应用

复合口服避孕药（COC）

作用机制
  • • 抑制排卵：阻断LH/FSH峰值
  • • 内膜抑制：减少内膜增生和前列腺素产生
  • • 宫颈黏液改变：增加粘稠度
  • • 子宫收缩减少：降低子宫肌肉兴奋性
适应人群
  • • 青年女性：需要避孕且有痛经
  • • 月经过多：合并月经量过多者
  • • NSAIDs失败：一线治疗效果不佳
  • • 长期管理：需要持续疼痛控制
注意事项
  • • 血栓风险：吸烟、高龄、肥胖者慎用
  • • 代谢影响：可能影响糖脂代谢
  • • 血压监测：定期检査血压变化
  • • 停药反应：停药后症状可能反弹

孕激素类药物

LNG-IUS (左炔诺孕酮宫内缓释系统)
  • • 局部作用：宫腔内药物浓度高，全身副作用小
  • • 内膜萎缩：渐进性内膜萎缩，减少月经量90%
  • • 有效期：5年，痛经缓解率达85-95%
  • • 适应症：子宫腺肌症、内膜异位症相关痛经
  • • 副作用：不规则出血(前6个月)、情绪改变
其他孕激素
地屈孕酮：

天然孕激素，10-20mg/日，周期第11-25天。副作用少，但需要长期服用。

甲羟孕酮：

150mg肌注每3个月。强效抑制排卵，但可能影响骨密度。

螺内酯：

抗雄激素作用，50-100mg/日。可减轻PMS症状。

GnRH激动剂

作用机制与适应症
  • • 机制：下调垂体受体，抑制卵巢功能
  • • 适应症：严重子宫内膜异位症相关痛经
  • • 效果：疼痛缓解率90-95%
  • • 用法：亮丙瑞林3.75mg肌注，每月1次
限制因素
  • • 使用期限：通常不超过6个月
  • • 副作用：更年期样症状、骨密度下降
  • • 回补治疗：需要激素回补减少副作用
  • • 停药后：症状可能复发

药物相互作用与监测要点

重要药物相互作用

NSAIDs相互作用
  • • 抗凝药：增加出血风险，需要监测凝血功能
  • • 利尿剂：可能减弱利尿效果，影响血压控制
  • • ACE抑制剂：可能影响肾功能和血压
  • • 锂盐：增加锂中毒风险
激素药物相互作用
  • • 抗生素：某些联用例抗生素可能影响避孕效果
  • • 抗癫痫药：可能影响激素代谢
  • • 圣约翰草：可能降低激素药物效果
  • • 抗真菌药：某些类型可能有相互作用

监测与随访要点

定期监测项目
  • • 疼痛评估：VAS评分，每月经周期记录
  • • 胃肠道：消化道症状，大便潜血
  • • 肝肾功能：长期用药者每3-6个月检查
  • • 血压监测：激素用药者每3个月
治疗性监测
  • • 药效评估：用药3个周期后综合评价
  • • 剂量调整：根据效果和耐受性调整
  • • 换药时机：单一药物无效时的选择
  • • 停药计划：长期用药的停药计划

继发性痛经专题：常见病因深度解析

继发性痛经概述

继发性痛经虽然只占痛经患者的5-10%，但由于其背后往往存在可能影响生育和生活质量的器质性疾病， 需要特别关注。根据 ACOG 和 WHO 的统计数据， 以下三种疾病是引起继发性痛经的最主要原因。

🎯 子宫内膜异位症 (Endometriosis)

疾病特征与流行病学

定义与分类

子宫内膜异位症是指功能性子宫内膜组织（腺体和间质）出现在子宫腔以外的部位， 最常见的部位包括卵巢、输卵管、盆腔腹膜、泌尿道和肠道。

  • • 腹膜病灶：最轻型，疼痛可能很严重
  • • 卵巢内膜异位囊肿：巧克力囊肿
  • • 深部浸润性内异症：最严重，影响生活质量
  • • 腺肌症：内膜组织入侵子宫肌层
流行病学数据
  • • 整体患病率：育龄女性6-10%
  • • 痛经女性中：38-87%（診断率依检查方法而异）
  • • 不孕女性中：20-50%
  • • 慢性盆腔痛：71-87%
  • • 发病年龄：25-35岁为高峰

痛经机制与特征

疼痛产生机制
  • • 周期性出血：异位内膜组织月经期出血
  • • 局部炎症：血液刺激引起炎症反应
  • • 粘连形成：反复炎症导致组织粘连
  • • 神经侵犯：病灶侵犯神经组织
  • • 前列腺素增加：病灶产生大量致痛介质
典型疼痛特征
  • • 时间：月经前1-2周开始，经期达高峰
  • • 性质：深部钝痛、刀割样或烧灼感
  • • 位置：下腹深部、腰骶部、直肠
  • • 放射：可放射至大腿内侧、肛门
  • • 程度：中重度，进行性加重

诊断要点与伴随症状

经典三联症
  • • 进行性痛经：77-87%患者
  • • 深部性交痛：62-77%患者
  • • 不孕症：30-50%患者
其他常见症状
  • • 慢性盆腔痛：非经期也有疼痛
  • • 月经异常：量多、不规律出血
  • • 胃肠道症状：便秘、腹胀、恶心
  • • 泌尿系症状：尿频、排尿痛
诊断金标准
  • • 腹腔镜：直视下确diagnosis病灶
  • • 组织活检：病理学确诊
  • • MRI：非侵入性诊断工具
  • • CA-125：辅助指标，>35U/ml提示

🔷 子宫肌瘤 (Uterine Fibroids)

疾病分类与流行病学

肌瘤类型分类
  • • 黏膜下肌瘤：向宫腔内生长，5-10%，症状最严重
  • • 肌壁间肌瘤：位于子宫肌层，70-80%，最常见
  • • 浆膜下肌瘤：向腹腔内生长，15-20%，症状相对轻
  • • 混合型：同时累及多个层次
流行病学特征
  • • 患病率：育龄女性20-30%
  • • 症状性肌瘤：25-35%需要治疗
  • • 高风险人群：35-45岁年龄组
  • • 种族差异：黑人女性患病率更高
  • • 家族史：有家族史者风险增加2-3倍

痛经产生机制

直接机制
  • • 宫腔变形：影响子宫收缩协调性
  • • 经血梗阻：经血引流不畅增加压力
  • • 局部缺血：肌瘤压迫血管引起缺血
  • • 炎症反应：肌瘤坏死引起炎症
间接效应
  • • 内膜面积增加：前列腺素产生增多
  • • 血管生成：异常血管网络形成
  • • 神经压迫：较大肌瘤压迫神经
  • • 激素影响：局部激素水平改变

症状特征与诊断要点

疼痛特征
  • • 痉挛性疼痛：尤其黏膜下肌瘤
  • • 持续性胀痛：较大肌瘤
  • • 下坠感：浆膜下肌瘤
  • • 腰骶部酸痛：后壁肌瘤
伴随症状
  • • 月经量增多：80-90%的患者
  • • 经期延长：>7天
  • • 压迫症状：尿频、便秘
  • • 不孕：5-10%的原因
诊断方法
  • • 妇科检查：子宫增大、质硬
  • • 超声检查：首选检查方法
  • • MRI：准确显示位置大小
  • • 宫腔镜：黏膜下肌瘤诊断
治疗原则
  • • 药物治疗：GnRH激动剂
  • • 介入治疗：动脉栓塞
  • • 手术治疗：肌瘤剔除
  • • 观察随访：无症状小肌瘤

🔶 子宫腺肌症 (Adenomyosis)

疾病定义与特征

医学定义

子宫腺肌症是指子宫内膜腺体和间质侵入子宫肌层，引起子宫均匀性增大、 月经量增多和进行性痛经的疾病。本质上是"内在性子宫内膜异位症"。

  • • 弥漫性：累及整个子宫肌层
  • • 局限性：腺肌瘤，局限性病灶
  • • 混合型：同时存在两种类型
流行病学与风险因素
  • • 患病率：育龄女性5-15%
  • • 高发年龄：35-50岁
  • • 经产妇：风险高于未产妇
  • • 家族史：有遗传倾向
  • • 激素暴露：长期雌激素刺激

痛经机制与症状

疼痛产生机制
  • • 肌层出血：异位内膜组织周期性出血
  • • 肌纤维破坏：内膜侵犯破坏肌纤维
  • • 局部炎症：慢性炎症反应
  • • 肌肉痉挛：异常肌肉收缩
  • • 神经侵犯：疼痛神经受损
特征性症状
  • • 进行性痛经：90%以上患者
  • • 月经量增多：70-80%患者
  • • 慢性盆腔痛：60%患者
  • • 性交痛：30-40%患者
  • • 压迫症状：子宫增大引起
诊断要点
  • • MRI：诊断金标准
  • • 超声：子宫增大，回声不均
  • • 妇检：子宫增大，质硬
  • • CA-125：可能轻度升高
鉴别诊断
  • • 子宫肌瘤：MRI可鉴别
  • • 内膜异位症：临床表现类似
  • • 内膜增生：病理检查鉴别
  • • 恶性肿瘤：年龄、影像特点
治疗策略
  • • LNG-IUS：一线治疗选择
  • • GnRH激动剂：短期症状控制
  • • 子宫切除：根治性治疗
  • • 胚胎保护：年轻患者考虑
