# Pull Request 创建指南

## 🎯 GitHub Pull Request 创建步骤

### 1. 访问GitHub仓库
打开浏览器，访问：`https://github.com/haha321-haha/hub-latest`

### 2. 创建Pull Request
1. 点击 **"Pull requests"** 标签
2. 点击 **"New pull request"** 按钮
3. 选择分支：
   - **Base branch**: `main`
   - **Compare branch**: `feature/dysmenorrhea-assessment-tool`

### 3. 填写PR信息
